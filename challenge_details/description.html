This challenge comes from the paper Towards Realistic Single-Task Continuous Learning Research for NER. The paper presents a method of splitting the StackOverflowNER dataset into 5 train/test episodes to create a more realistic and challenging continual learning setting.

This challenge involves the "skewed" episodes from the paper. You should train a model on each of the 5 training episodes in succession, without revisiting prior episodes. You may employ any continual learning strategy you wish (including replay of small amounts of data from previous episodes). You must evaluate the final model (trained on all 5 episodes) on all 5 test episodes. You will upload a submission file in json format containing your predicted tags for each example in each episode. These predictions will be evaluated against the ground truth tags.

To download the full train/test data and see more information about the dataset construction, visit the StackOverflowNER-NS Github repository.
